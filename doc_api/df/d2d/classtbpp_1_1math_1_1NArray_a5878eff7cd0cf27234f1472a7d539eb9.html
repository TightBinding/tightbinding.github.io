template<typename T, size_t N>
void tbpp::math::NArray< T, N >::save_dump ( const std::string &  filename) const
inline

Quickly save the contents to the disk in binary format.

This function is designed to quickly save the array to the disk. The array is saved in native endianness and no attempt is made to convert endianness.

Warning
Use save_ascii if the file is intended to be shared between computers.
