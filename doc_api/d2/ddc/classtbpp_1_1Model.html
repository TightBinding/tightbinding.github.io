Detailed Description

A class representing a tight-binding Hamiltonian.

The Model class is the central class representing a tight-binding Hamiltonian. The class is usually initialized by using a Lattice class but can also be manually initialized.

Once initialized, the class provides methods for determining the Hamiltonian matrix at each k-point as well as derivatives of the Hamiltonian.

Get the derivative of Hamiltonian along x,y,z.

Note that k1,k2,k3 are in reduced coordinates. This means that the first BZ is from -0.5 < k1,k2,k3 <0.5.

If dx,dy or dz is NULL, then the derivative is not computed along that axis.
