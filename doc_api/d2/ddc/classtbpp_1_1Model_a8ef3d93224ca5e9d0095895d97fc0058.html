Get the Hamiltonian at a specific k-point.

Note that k1,k2,k3 are in reduced coordinates. This means that the first BZ is from -0.5 < k1,k2,k3 <0.5.

Parameters
[out]dPointer to array of size states*states
[in]k1k-point in reduced coordinates along b1
[in]k2k-point in reduced coordinates along b2
[in]k3k-point in reduced coordinates along b3
