◆ EHFile()

tbpp::EHFile::EHFile ( const std::string &  filename,
const std::string &  mode = "r" 
)

Create a new EHFile.

Possible values for mode,

Mode Description
r Open in read only. If file does not exist throw exception.
r+ Open in read-write. If file does not exist throw exception.
w Open in read-write. If file already exists overwrite.
x Create file. If it already exists, throw exception.

Default behavior is r open file in read-only.

Parameters
filenamePath to the file to open or create
modeMode to open file in see above for possible modes
Exceptions
invalid_argumentIf mode is invalid
H5::FileIExceptionIf a file error occurs
