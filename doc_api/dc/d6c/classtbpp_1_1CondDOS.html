Detailed Description

Compute DOS and DC electrical conductivity using self-energy.

The CondDOS class allows for computing the Density of States and zero temperature DC electrical conductivity using Green's function methods and the self-energy returned by the CPA module. The electrical conductivity is determined using the Kubo-Greenwood formalism.
