Detailed Description

Contains a collection of nodes representing a simulation pipeline.

The Context class contains a collection of nodes which are identified by a unique name and facilitates saving a loading the simulation pipeline. The class allows for connections between nodes to be serialized by saving the names of connected nodes and enables connections to be restored when loading by generating all nodes before initializing them.
